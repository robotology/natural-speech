Welcome to Yarp.js

This is a bundle of sample applications that show how YARP can be combined with modern Web tools and APIs using yarp.js.

You can directly use these applications in your system in order to include functionalities such as speech recognition or 3D data visualization or you can adapt them to your needs by just writing few lines of code.

Note: before starting please be sure to have a yarp server running on your network.
